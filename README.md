# Sales & Profit Analysis Dashboard (Power BI)

## Author

Tanisha Arora

---

## Project Overview

This project is an interactive Power BI dashboard built using the Global Superstore dataset. It focuses on analyzing sales, profit, customer behavior, and regional performance to generate meaningful business insights.

---

## Objective

* Transform raw data into actionable insights
* Analyze sales and profit performance across different regions
* Identify top-performing countries, cities, and customers
* Support data-driven decision-making

---

## Tools & Technologies

* Power BI
* Data Visualization Techniques

---

## Dashboard Features

* KPI cards for Total Sales, Total Profit, Average Order Value, and Total Customers
* Profit Margin Gauge to evaluate overall profitability
* Top 5 Countries by Profit (Pie Chart)
* Top Countries by Sales (Bar Chart)
* City-wise Orders vs Quantity Analysis
* Top Customers by Sales
* Sales vs Profit by Market (Time-based analysis)
* Interactive filters (Region, Category, Year)

---

## Key Insights

* The United States is the top-performing country contributing the highest sales and profit.
* Total Sales (~12.64M) are significantly higher than Total Profit (~1.47M), indicating moderate profit margins.
* The average order value (~504.99) shows consistent customer purchasing behavior.
* A small group of customers contributes a large portion of total revenue.
* Certain cities have higher order quantities, indicating strong regional demand.
* Profit margin (~0.12) suggests potential for improving pricing or reducing costs.

---

## Conclusion

This project demonstrates how data visualization tools like Power BI can be used to transform complex datasets into clear and actionable insights for better decision-making.
